# vd-scraping

A web scraper for Github repositories created in Python with BeautifulSoup.

## Requirements:
- Python 2.7
- Pip 10.0.1
- BeautifulSoup 4.4.0

## How to run:

- First install all dependencies with `make setup`
- Make sure the `repositories.txt` follows this format:
```
jlgm/QuakeParser
jlgm/vd-scraping
```
- Run with `make start`
